## Luồng campaign hiện tại

Đây là phần rất quan trọng đã chỉnh lại đúng nghiệp vụ:

1. User tạo campaign từ UI.
2. Không cho nhập tay tên channel nữa.
3. Bắt buộc chọn từ danh sách `TelegramGroup` đã sync.
4. UI gửi `telegramGroupId` lên backend.
5. Backend tạo record `Campaign`.
6. Sau đó backend gọi Telegram API `createChatInviteLink` thật.
7. Nếu Telegram tạo link thành công:
   - update `campaign.inviteCode`
   - lưu mapping invite link theo campaign/group
8. Nếu Telegram fail:
   - rollback/xóa campaign
   - không để lại bản ghi rác

File chính:

- `apps/api/src/campaigns/campaigns.controller.ts`
- `apps/api/src/campaigns/campaigns.module.ts`
- `apps/api/src/campaigns/campaigns.service.ts`
- `apps/api/src/telegram/telegram.module.ts`
- `apps/web/src/components/platform-dashboard.tsx`

## Secret và bot token đang lưu thế nào

- Hiện chỉ hỗ trợ 1 bot token active.
- Token được lưu ưu tiên trong DB dưới key `telegram.bot_token`.
- Nếu DB chưa có thì fallback sang env `TELEGRAM_BOT_TOKEN`.
- Token trong DB được mã hóa AES-256-GCM.
- Metadata bot lưu trong bảng singleton `TelegramBotConfig`.
